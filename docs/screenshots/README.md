# Captures d'écran

Remplacez ce dossier par de vraies captures (PNG), référencées dans le README :

- `chat.png`        — onglet Chat (vue d'accueil ou une conversation)
- `prompts.png`     — onglet Prompts (bibliothèque + bouton Nouveau prompt)
- `diagnostic.png`  — onglet Diagnostic (état + section Consommation tokens)
- `checkpoints.png` — onglet Runs (panneau Checkpoints fichiers)

⚠️ Vérifiez qu'aucune capture ne montre d'information personnelle
(chemins C:\Users\..., noms de sessions nominatifs, etc.).
